Personal Voice Assistant 🎙️

A Python-based Personal Voice Assistant that listens to voice commands, converts speech into text, processes the command, and responds using speech.

This project demonstrates the basic working concept of modern voice assistants such as Google Assistant and Amazon Alexa.

Features

🎤 Speech-to-Text conversion

🔊 Text-to-Speech response

🌐 Open websites like YouTube and Google

⏰ Tell the current time

💬 Repeat user commands

⚠ Basic error handling

Technologies Used

Python

SpeechRecognition

pyttsx3

Webbrowser module

Datetime module

Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/Personal-Voice-Assistant.git

Go to the project folder:

cd Personal-Voice-Assistant

Install required libraries:

pip install SpeechRecognition
pip install pyttsx3
pip install pyaudio
How to Run

Run the Python file:

python assistant.py

The assistant will start listening for voice commands.

Sample Commands

You can try commands like:

Hello
Open YouTube
Open Google
What is the time
